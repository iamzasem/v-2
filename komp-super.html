Comparative and Superlative Before Nouns in German

A guide to using comparative (-er) and superlative (-st/-est) adjectives before nouns, with correct endings for gender, number, and case, including examples, exercises, and resources.

What Are Comparative and Superlative Before Nouns?

These forms compare (more/less) or indicate extremes (most/least) for nouns, requiring adjective endings that match the noun’s gender, number, and case.

Key Points

  • Comparative: Add -er (e.g., schneller - faster).
  • Superlative: Add -st/-est (e.g., schnellst- - fastest).
  • Endings: Vary by article and case (e.g., -e, -es, -en).

Structure and Formation

Formation

Article + adjective (comparative/superlative) + noun, with endings adjusted for case.

Uses of Comparative and Superlative

Basic: Comparison

  1. Ein größerer Tisch (A bigger table)

Intermediate: Extreme Quality

  1. Die kleinste Katze (The smallest cat)

Advanced: Description

  1. Ein jüngerer Bruder (A younger brother)

Exercises to Practice Comparative and Superlative

Exercise 1: Comparative Before Nouns

Select the correct comparative form.

  1. Ein ___ Hund (schnell)
  2. Eine ___ Tasche (klein)
  3. Ein ___ Stuhl (hoch)
  4. Das ___ Fenster (groß)
  5. Eine ___ Jacke (warm)

Answers:

  1. Ein schneller Hund
  2. Eine kleinere Tasche
  3. Ein höherer Stuhl
  4. Das größere Fenster
  5. Eine wärmere Jacke

Exercise 2: Superlative Before Nouns

Choose the correct superlative form.

  1. Der ___ Baum (hoch)
  2. Die ___ Zeit (schnell)
  3. Das ___ Haus (klein)
  4. Eine ___ Blume (schön)
  5. Der ___ Freund (alt)

Answers:

  1. Der höchste Baum
  2. Die schnellste Zeit
  3. Das kleinste Haus
  4. Eine schönste Blume
  5. Der älteste Freund

Exercise 3: Mixed Comparative and Superlative

Choose the correct form.

  1. Ein ___ Tisch (groß)
  2. Die ___ Katze (klein)
  3. Das ___ Lied (leise)
  4. Eine ___ Schwester (jung)
  5. Der ___ Stein (schwer)

Answers:

  1. Ein größerer Tisch
  2. Die kleinste Katze
  3. Das leisere Lied
  4. Eine jüngere Schwester
  5. Der schwerere Stein

Exercise 4: Translate

Choose the correct translation.

  1. A taller tree
  2. The smallest house
  3. A prettier flower
  4. The fastest car
  5. A younger brother

Answers:

  1. Ein höherer Baum
  2. Das kleinste Haus
  3. Eine schönere Blume
  4. Das schnellste Auto
  5. Ein jüngerer Bruder

Exercise 5: Correct the Mistake

Choose the corrected sentence.

  1. Ein groß Tisch
  2. Die schnell Zeit
  3. Das klein Haus
  4. Eine schön Blume
  5. Der alt Freund

Answers:

  1. Ein größerer Tisch
  2. Die schnellste Zeit
  3. Das kleinste Haus
  4. Eine schönere Blume
  5. Der älteste Freund

Tips to Master Comparative and Superlative

  • Learn Endings: Match endings to gender, case, and article (e.g., -er, -e).
  • Umlaut Check: Add umlauts where needed (e.g., alt → älter).
  • Article Matters: Use ein for indefinite, der for definite.
  • Practice Pairs: Compare kleiner vs. kleinste with nouns.
