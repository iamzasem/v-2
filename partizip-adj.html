Partizip als Adjektiv (Participles as Adjectives)

A guide to using present (Partizip I) and past (Partizip II) participles as adjectives in German, with forms, examples, exercises, and resources.

Understanding Participles as Adjectives

Definition

Participles (Partizip I and II) describe nouns like adjectives, adding action or state details, e.g., laufend (running), geschrieben (written).

Formation

- Partizip I: Verb stem + -end (e.g., laufen → laufend).
- Partizip II: Regular: ge- + stem + -t/et (e.g., machen → gemacht); Irregular: ge- + unique form (e.g., sehen → gesehen).

Endings: Declined like adjectives (e.g., der laufende Hund).

Rules

- Before noun.
- Matches gender, number, case.
- Partizip I: Active (bellender Hund).
- Partizip II: Passive/State (gebautes Haus).

Uses

1. Action: Ein singendes Mädchen.
2. State: Ein geöffnetes Fenster.
3. Characteristics: Ein lachender Mann.
4. Temporary: Ein schlafendes Baby.

Exercises to Practice

Exercise 1: Identify Partizip I or II

Select the correct participle type.

  1. Ein laufender Hund
  2. Ein geschriebenes Buch
  3. Die singenden Kinder
  4. Das geöffnete Fenster
  5. Ein arbeitender Mann

Answers:

  1. Partizip I
  2. Partizip II
  3. Partizip I
  4. Partizip II
  5. Partizip I

Exercise 2: Form the Participle

Choose the correct participle form.

  1. lachen → ___ Mann (laughing man)
  2. bauen → ___ Haus (built house)
  3. schreiben → ___ Frau (writing woman)
  4. sehen → ___ Film (seen movie)
  5. spielen → ___ Kinder (playing children)

Answers:

  1. Ein lachender Mann
  2. Ein gebautes Haus
  3. Eine schreibende Frau
  4. Ein gesehener Film
  5. Die spielenden Kinder

Exercise 3: Add the Correct Ending

Choose the correct adjective ending.

  1. Ein ___ Hund bellt. (laufend, masculine nominative)
  2. Die ___ Tür ist zu. (geschlossen, feminine nominative)
  3. Das ___ Kind schläft. (spielend, neuter nominative)
  4. Ich sehe die ___ Frauen. (tanzend, plural accusative)
  5. Mit dem ___ Mann spreche ich. (arbeitend, masculine dative)

Answers:

  1. Ein laufender Hund bellt.
  2. Die geschlossene Tür ist zu.
  3. Das spielende Kind schläft.
  4. Ich sehe die tanzenden Frauen.
  5. Mit dem arbeitenden Mann spreche ich.

Exercise 4: Translate

Choose the correct translation.

  1. A running dog is fast.
  2. The written letter is long.
  3. The singing birds are loud.
  4. A built house is expensive.
  5. The laughing girl is happy.

Answers:

  1. Ein laufender Hund ist schnell.
  2. Der geschriebene Brief ist lang.
  3. Die singenden Vögel sind laut.
  4. Ein gebautes Haus ist teuer.
  5. Das lachende Mädchen ist glücklich.

Exercise 5: Mixed Examples

Choose the correct form.

  1. Ein ___ Kind weint. (schreiend, neuter nominative)
  2. Die ___ Fenster sind neu. (geöffnet, plural nominative)
  3. Ich mag den ___ Mann. (lachend, masculine accusative)
  4. Eine ___ Frau singt. (tanzend, feminine nominative)
  5. Das ___ Essen schmeckt gut. (gekocht, neuter nominative)

Answers:

  1. Ein schreiendes Kind weint.
  2. Die geöffneten Fenster sind neu.
  3. Ich mag den lachenden Mann.
  4. Eine tanzende Frau singt.
  5. Das gekochte Essen schmeckt gut.

Tips to Master Participles as Adjectives

  • Partizip I: Add -end for active actions (e.g., laufend).
  • Partizip II: Use ge- forms for states (e.g., gemacht).
  • Endings: Match article and case (e.g., der laufende, ein laufender).
  • Practice: Start with common verbs (laufen, schreiben, bauen).
